JavaScript vs. Ruby

The Basics

JAVASCRIPT

Defining a variable: 


      var myVariable=2;

Converting a string to a number: 


      Number("2");

While loops: 


      var n=1
      while (n<5){
        console.log(n);
        n++;
        }
Take note of the semicolons, parenthesis and brackets. Also notice you increment the counter with ++ instead of +=.

For loops: 


      for (var i=0; i<=5; i++){
        console.log(i);
        }

Defining a class: 


      function Person (){}

New instances of objects: 


      var person1= new
      Person();
      var person2= new
      Person();

The constructor method:
This is called at the moment of instantiation and there is no need to explicitly define it. 


    function Person(){
    alert('Someone new.');
    }
    var person1= new
    Person();
    var person2= new
    Person();

Properties:
These are variables available to instances of the class (function.) 'This' refers to the current object, which is similar to the concept of 'self' in Ruby. 


    function Person(name){
      this.name=name;
      alert('Someone new.');
      }

      var person1= new
      Person('Tom');
      var person2= new
      Person('Betty');

Methods:
Methods follow the same logic as properties except they are functions. To define a method, assign a function to a named property of the class's prototype property. 


      function Person(name){
      this.name=name;
      alert('Someone new.');
      }

      Person.prototype.sayHello
      =function (){
        alert('hello');
      };

      var person1= new
      Person('Tom');
      var person2= new
      Person('Betty');

      person1.sayHello();
      //-> hello

RUBY

Defining a variable: 


      myVariable=2;

Converting a string to a number: 


      "2".to_i

While loops: 


    n=1
    while n <= 5
      puts n
      n+=1
    end




For loops: 


    for n in 1..5
      puts n
      n+=1
    end

Defining a class: 


    class Person
    end

New instances of objects: 


    @person1=Person.new
    @person2=Person.new

The initialize method:
It's called automatically at the moment of instantiation 


    class Person
      def initialize
        puts "Someone new."
      end
    end

    @person1=Person.new
    @person2=Person.new

Instance Variables:
These are variables to instances of the class. Defined with an @. 


    class Person
      def initialize(name)
        @name=name
        puts "Someone new."
      end
    end

    @person1=Person.new('Tom')
    @person2=Person.new('Betty')

Methods:
Methods are called the same way that instance variables are called on an object. However, they are defined within the class. 


  class Person
    def initialize(name)
      @name=name
      puts "Someone new."
    end

    def say_hello
      puts "hello"
    end
  end

    @person1=Person.new('Tom')
    @person2=Person.new('Betty')

    @person1.say_hello
      //-> hello
