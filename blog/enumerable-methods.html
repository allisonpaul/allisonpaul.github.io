The map method.

March 12, 2016

What is an Enumeral?

Well, good question. An enumeral is a mixin, or a module that consists of a bunch of methods packaged together. Enumeral provides methods for collections of items (think arrays, hashes, and the like) so you can iterate through them. This module contains many useful methods like #map, #group_by, #select , and #cycle . I suggest checking out the Ruby Docs for a full list. In this post we are going to talk about the map method.

Lets Map.

Say you want to keep all the items in your object and modify them somehow? #map will run your code block and give you an object filled with whatever your block returned. This method is also referred to by#collect. Map is useful when you want to change every value in an enumerable.

With #each 


              multiply = []
              [2,4,6,8].each do |n|
                multiply << n * 5
              end

With #map


              [2,4,6,8].map do |n|
                n*5
              end

Map returns a new array with the results of the code block, so it's not neccesary to shovel into a new array like you do with #each. If you would rather alter the original array, you can turn #map into a destructive method by adding a bang operator (!) like so map!.
